# Code of Conduct

This repository follows a simple, minimal code of conduct that aims to keep discussion constructive, respectful and focused on technical clarity.

## Expectations

- Be respectful to all contributors.
- Critique ideas, not people.
- Maintain relevance to the UBOP documentation and historical context.
- Use clear and neutral language.
- Help others understand design decisions.

## Unacceptable Behavior

- Personal attacks or insults.
- Harassment or discrimination of any kind.
- Derailing discussions with unrelated topics.
- Aggressive promotion of unrelated technologies.
- Attempts to pressure maintainers into adding new UBOP features.

## Scope

This code of conduct applies to:
- GitHub issues  
- Pull requests  
- Discussions  
- Any messaging associated with this repository  

## Enforcement

Maintainers may:
- remove inappropriate content  
- block or limit participation  
- close discussions not aligned with the purpose of the archive  

If you observe unacceptable behavior, open a GitHub Issue titled "Code of Conduct Violation" with a factual description.
